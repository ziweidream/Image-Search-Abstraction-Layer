Image Search Abstraction Layer

User stories

  • I can get the image URLs, alt text and page urls for a set of images relating to my search string.

    Example: https://jazzy-xylophone.glitch.me/image/cute cats

  • I can paginate through the responses by adding a ?offset=5 parameter to the URL.

    Example: https://jazzy-xylophone.glitch.me/image/cute%20cats?offset=5

  • I can get a list of the most recently submitted search strings.

    Example: https://jazzy-xylophone.glitch.me/latest
